Tecnologías de Web APIs

son interfaces que permiten la comunicación entre diferentes sistemas a través de la web.

Tipos de Web APIs

RESTful

Basadas en el estilo arquitectónico REST (Representational State Transfer). Son las más comunes, utilizando métodos HTTP (GET, POST, PUT, DELETE, etc.) para interactuar con recursos.

GraphQl

Un sistema de consulta y manipulación de datos desarrollado por Facebook. Permite a los clientes especificar exactamente qué datos necesitan.

SOAP

Un protocolo basado en XML para intercambio de información. Aunque más antiguo, todavía se utiliza en aplicaciones empresariales.
